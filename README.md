# CS330

I tackle software development one small step at a time. When faced with a large project, I utilize incremental coding to divide it into smaller, more manageable components. I've discovered that when designing anything, whether it's a 3D scene or software, it's essential to view it from multiple perspectives to understand how everything fits together and to identify elements that may not seem significant until later stages. The design process I adhere to includes Ask, Imagine, Plan, Create, and Improve. Initially, I ask myself what I aim to accomplish with the project, then I visualize what the scene will look like. Next, I devise a plan to address specific objects by simplifying them into basic shapes, followed by coding to implement my design, and finally, I dedicate time to enhancing the overall appearance of the scene.

As mentioned earlier, this design process can be applied to any software project you undertake, and when you reach the coding phase, you simply break it down into smaller steps.

Computer science can significantly aid me in reaching my goals by providing a set of powerful skills and tools that enhance my problem-solving, creativity, and analytical abilities. Designing visual content fosters creativity, helps in data driven decision making, and will add another skill set that can give me an edge in my career.
